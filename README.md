# Vysakh T - Portfolio

Minimalist portfolio built with React, Vite, Tailwind CSS, and shadcn-ui.

## Tech Stack
- React 18
- TypeScript
- Vite
- Tailwind CSS
- shadcn UI

## Getting Started

```bash
# Install dependencies
bun install

# Start development server
bun run dev

# Build for production
bun run build
```
